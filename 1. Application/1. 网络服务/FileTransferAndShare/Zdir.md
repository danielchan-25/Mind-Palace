## Windows

下载解压后，使用 `cmd` 进入到 `zdir` 目录，进行初始化
```powershell
D:\zdir>zdir.exe init
Init success.

# 如果出现：Initialization failed, the configuration file already exists.
# 建议把目录删掉并重装
```
初始化成功后，进入 **服务**，找到 **Zdir**，将它启动即可。
